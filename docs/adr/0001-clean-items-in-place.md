# 直接修改 Zotero 条目字段

BibTeX Clean 插件最初只提供"复制清理后的 BibTeX"功能——读取选中条目、应用清理规则、输出到剪贴板，是一个只读辅助工具。在 2026-07-02 的设计讨论中，我们决定将核心操作改为直接修改 Zotero 条目本身的字段：右键选中条目后，通过确认对话框展示将要发生的变更，确认后把清理后的值写回条目。

做出这个决定的原因是插件的核心价值从"导出干净的 BibTeX"转向"帮助用户整理条目元数据"。直接修改条目能让清理结果持久化，避免用户每次导出都要重复清理。为了降低破坏性写操作的风险，我们同时引入了确认对话框（先模拟运行再展示 diff）和仅当前会话有效的"撤销上次清理"功能。

被拒绝的替代方案包括：保持只读导出（无法持久化清理结果）、创建新条目副本（会污染用户的资料库）、以及无确认直接写入（风险过高）。
