# Triage Labels

技能使用五种标准分流角色进行交流。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库 tracker 中的标签 | 含义                            |
| -------------------------- | ----------------------- | ------------------------------- |
| `needs-triage`             | `needs-triage`          | 维护者需要评估该 issue          |
| `needs-info`               | `needs-info`            | 等待报告者补充信息              |
| `ready-for-agent`          | `ready-for-agent`       | 已完全明确，可由离席 agent 处理 |
| `ready-for-human`          | `ready-for-human`       | 需要人工实现                    |
| `wontfix`                  | `wontfix`               | 不会处理                        |

当技能提到某个角色（例如“应用 AFK-ready 分流标签”）时，使用本表对应行的标签字符串。

编辑右侧列以匹配你实际使用的词汇。
